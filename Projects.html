SE&A project

Voor dit project moesten wij een quiz maken voor studenten maar de studenten mochten niet gebruik maken van hun privé toestellen zoals laptop of gsm.

Wij hadden als oplossing een IOT toestel waarmee je alleen kon antwoorden op vragen en niks anders.
Voor dit project was ik verrantwoordelijk voor de quiz zelf de frontend en het bijbehorende javascript. Aangezien docenten een quiz konden aanmaken, een bestaande quiz aanpassen en delen met andere docenten.

Uit dit project heb ik geleerd dat hoe handig IOT toestellen kunnen zijn.

Project Hosting

Voor dit project moesten wij een hosting platform maken. Waar de eingebruiker een php webaplicatie op kan gaan hosten.
Hiervoor hebben wij hardware gekregen die wij moesten configureren om het platform zelf op te draaien. De setup bevatte: 3 ESXi servers, 2 storage servers, een firewall en een switch.

Voor dit project heb ik geholpen met de hardware te configureren al was het vaak meekijken hoe het moest en vervolgens zelf doen. Ook heb ik De frontend van het platform ontwikkeld samen met de Api voor ons project.

Tijdens het maken van het project heb ik heel veel geleerd over servers en hoe je deze configureert.

Linux Webservices

Voor dit project heb ik Linux Webservices gebruikt met Docker om een Apache2 en MySQL container op te zetten voor het hosten van een webpagina. Met behulp van Nginx heb ik deze pagina geschaald. Dit was een nieuwe uitdaging voor mij, maar dankzij verplichte AI-ondersteuning kon ik het project met succes afronden.

In mijn verslag heb ik alle gebruikte commando's en interacties met ChatGPT gedocumenteerd, waardoor ik mijn kennis van de technologieën kon uitbreiden. Het eindresultaat was een eenvoudige HTML-pagina met dynamische gegevens uit een MySQL-database. Bij het schalen verdeelde ik de belasting over twee Apache2-containers.

Dit project heeft mijn begrip van SQL- en Apache2-containers vergroot en heeft me ook inzicht gegeven in het schalen van webservers.
